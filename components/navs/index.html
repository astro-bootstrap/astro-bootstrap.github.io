Navs

Documentation and examples for how to use Bootstrapâ€™s included navigation components.

View the official docs here

The base .nav component provides a strong foundation for building all types of navigation components. For that reason an astro component has not been included.

Examples

Basic

  • Home
  • Link
  • Active
  • Disabled
Astro file
---
// Template example only
import { ActiveLink } from 'astro-bootstrap';
const navs = [
  { href: '/', text: 'Home' },
  { href: '#', text: 'Link' },
  { href: '/components/navs', text: 'Active' },
  { href: '/hidden', text: 'Disabled', disabled: true },
];
---

<ul class="nav">
  {
    navs.map((nav) => (
      <li class="nav-item">
        <ActiveLink href={nav.href} class="nav-link" disabled={nav.disabled}>
          {nav.text}
        </ActiveLink>
      </li>
    ))
  }
</ul>
