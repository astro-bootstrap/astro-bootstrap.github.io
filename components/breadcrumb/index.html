API

Breadcrumb

import { Breadcrumb } from 'astro-bootstrap'
NameTypeDefaultDescription
aria-label string Breadcrumb Set the aria-label property value on the "breadcrumb" div.
id string null Set the id on the "breadcrumb" div.
class string null Set the class on the "breadcrumb" div.
itemClass string null Set the class on the "breadcrumb-item" div.
