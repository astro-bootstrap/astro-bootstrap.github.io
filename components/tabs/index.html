Tabs

Make sure you set one of your Tabs to active via your data.

Tab 1 text
Tab 2 text
Tab 3 text
Astro file
---
import { Tabs } from 'astro-bootstrap';
const tabs = [
  { title: 'Tab 1', body: 'Tab 1 text', active: true },
  { title: 'Tab 2', body: 'Tab 2 text' },
  { title: 'Tab 3', body: 'Tab 3 text' },
];
---

<Tabs {tabs} id="exampleTabs" class="p-3" fade />

id and class are optional
class adds a class to the tab content pane
omit fade if you don't want a fade transition between content panes
