Get Started Components

Hand crafted for Astro.js

To maintain astro's fast build times we have created native Astro components. To minimise JS sent to the browser, our components use stock bootstrap JS modules. No need to hydrate your component with React..

Bootstrap CSS at its core

Built with compatibility in mind, all our components are compadible with Bootstrap 5. By relying entirely on the Bootstrap stylesheet, Astro-Bootstrap components are compadible with existing themes that you have created. You can also customise with Bootstrap's powerfull SCSS variables..

Built for rapid development

We understand that excessive components will slow down development for a developer who is experienced with bootstrap. Since astro gives you such easy control over HTML we have only created Astro components in cases where they will make your code more dry. Stock bootstrap JS has been included..
